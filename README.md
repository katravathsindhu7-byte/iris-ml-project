# Iris ML Classification Project 🌸

## 📌 Overview
This project uses Machine Learning to classify iris flowers into different species based on their features.

## 🧠 Model Used
- Random Forest Classifier

## 📊 Dataset
- Iris dataset from scikit-learn
- Total samples: 150
- Classes:
  - Setosa
  - Versicolor
  - Virginica

- scikit-learn
